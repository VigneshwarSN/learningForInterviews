## üèóÔ∏è Phase 1: Foundations

### Week 1: Git & GitHub
*The industry standard for version control. You must know this cold.*
- **Official Book**: [Pro Git (Chapters 1-3)](https://git-scm.com/book/en/v2) - *Read this for deep understanding.*
- **Interactive Tutorial**: [Leaern Git Branching](https://learngitbranching.js.org/) - *Highly recommended for visualizing branches.*
- **Video Guide**: [Git & GitHub Crash Course (FreeCodeCamp)](https://www.youtube.com/watch?v=RGOj5yH7evk) - *Watch the first hour.*
- **Cheat Sheet**: [Git Cheat Sheet](https://education.github.com/git-cheat-sheet-education.pdf)

### Week 2: Advanced Python
- **Official Docs**: [Python 3 Tutorial](https://docs.python.org/3/tutorial/index.html) - *Focus on Data Structures and Listen Comprehensions.*
- **Deep Dive**: [Real Python](https://realpython.com/) - *Excellent articles on specific topics like Decorators and Generators.*
- **Practice**: [HackerRank Python Path](https://www.hackerrank.com/domains/python)

### Week 3: SQL
- **Interactive**: [SQLZoo](https://sqlzoo.net/) - *Hands-on practice queries.*
- **Tutorial**: [Mode SQL Tutorial](https://mode.com/sql-tutorial/) - *Great for data analysis focus.*
- **Practice**: [LeetCode SQL 50](https://leetcode.com/studyplan/top-sql-50/)

### Week 4: Applied AI/ML
- **Library**: [Scikit-Learn Getting Started](https://scikit-learn.org/stable/getting_started.html)
- **Course**: [Kaggle Learn: Intro to Machine Learning](https://www.kaggle.com/learn/intro-to-machine-learning) - *Very quick, practical modules.*
- **Pandas**: [Pandas User Guide (10 Minutes to pandas)](https://pandas.pydata.org/docs/user_guide/10min.html)
